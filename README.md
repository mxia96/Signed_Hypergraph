# Groupwise Signed Hypergraph Learning for Resting-State fMRI

Code for the paper *"Groupwise Signed Hypergraph Learning for Resting-State fMRI Analysis"* (Xia & Shi).

- `hypergraph_construction/` — signed hypergraph construction (NumPy)
- `model/` — hypergraph neural network (PyTorch)
- `configs/` — YAML configs
